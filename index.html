WHICH TRACK SHOULD I CHOOSE?

Toward the end of Intro to Programming, all students will need to decide which track they would like to pursue for their Epicodus education. While all languages have similarities, and switching between languages and platforms is very common in both internships and jobs, you should still choose the track you wish to complete with care

Answer the following questions to see what track best suits you!

Would you like to attend Epicodus in Seattle?

Do you want to develop back end applications?

Is creating content that the user directly interacts with exciting to you?

Do you want to work for a start up?

Do you want to develop for mobile devices?

Suggested Track:

Ruby, JavaScript, Rails

Based on the information we have gathered, the Ruby on Rails track would be the best fit for you! Below are brief descriptions of what each section will cover. If you have any more questions feel free to check out Epicodus.

Ruby

This course builds on the programming fundamentals from Introduction to Programming and gives students more difficult problems to solve, a more advanced set of tools to solve them with, and establishes the foundations for learning the real-world technologies used by programmers on the job.

JavaScript

This course builds on the programming fundamentals from Introduction to Programming and gives students more difficult problems to solve and a more advanced set of tools to solve them with. It also introduces students to the front-end MVC paradigm that powers many real-world web applications.

Since JavaScript is the only programming language understood by all web browsers, it's become one of the most popular languages and necessary for most web and mobile developers to know.

Rails

After establishing solid programming fundamentals in Ruby and JavaScript, this course introduces students to the Rails framework used to build interactive web applications.

Ruby and Rails are typically used by younger companies and startups, especially for building interactive web applications. Portland companies using Ruby and Rails who have hired Epicodus graduates include New Relic, LivingSocial, and SpendWell.
